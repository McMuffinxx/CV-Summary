# CV Summary Maker (Streamlit)

This repo is ready for Streamlit Cloud.

**Structure**
```
.
├─ cv_summary_app6_SUMMARY_ONE_ROLE_FIXED_TP13_HEADER_ONLY.py
├─ requirements.txt
├─ runtime.txt
└─ template/
   └─ CURRICULUM VITAE.docx   # Replace with your final template if needed
```

**Deploy**
1. Push this folder to a GitHub repo.
2. On https://share.streamlit.io/ → New app → select this repo.
3. File path: `cv_summary_app6_SUMMARY_ONE_ROLE_FIXED_TP13_HEADER_ONLY.py`
4. Set environment variables in app settings (optional but recommended):
   - `GEMINI_API_KEY`
   - `OPENAI_API_KEY` (optional)

The app title is **CV Summary Maker** as requested.
